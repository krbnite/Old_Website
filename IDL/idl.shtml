These libraries contain loads of helpful and convenient functions for working with data sets from ground-based magnetometers (INTERMAGNET, Antarctic AGOs), spaceborne instruments (the plasma and magnetic field instruments on ACE, the RBSPICE instrument on the Van Allen Probes), to polar boundary data provided by other researchers (e.g., Longden UV/FUV boundaries, Newell DMSP boundaries). More importantly, my Signal Analysis package has many useful analysis and visualization tools for time series and spectral analysis.
